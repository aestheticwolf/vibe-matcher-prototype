# Vibe Matcher Prototype

A semantic search demo that matches abstract user vibes (e.g., "cozy coffee shop") to fashion products using cosine similarity.

## Tech Used
- Python
- Pandas, scikit-learn, matplotlib
- Mock embeddings (for fast, local prototyping)

## How to Run
1. Open `vibe_matcher.ipynb` in [Google Colab](https://colab.research.google.com)
2. Run all cells
3. Try your own vibe queries!

> Built as a learning project to explore AI-powered personalization in retail.
